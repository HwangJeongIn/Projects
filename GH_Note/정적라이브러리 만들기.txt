정적라이브러리 만들기

1. win32 콘솔응용프로그램 or windows 데스크톱 마법사로 프로젝트 만든다.

2. 정적라이브러리로 지정

3. 소스파일과 해더파일 작성

4. 빌드
	Visual Studio Command line 에서 빌드하는 경우 2단계를 거쳐야 한다.
		1. obj파일이 만들어져야 한다.
		2. 링크하여 정적라이브러리를 만들어야한다. // lib 파일

	첫번째 과정은
	cl /c /EHsc 파일명.cpp 가 호출된다.
	이 의미는 cl명령을 사용하여 Cl.exe컴파일러를 호출하고
	/c옵션은 링킹없는 특정 컴파일한다. 이 과정을 통해 obj파일을 만들어 낸다.

	두번째 과정은
	lib 파일명.obj 가 호출된다.
	이는 코드를 링크하고 lib파일을 만들어 낸다.

5. 사용
빌드과정이 끝나면 lib와 pdb(debug모드일때 디버그 정보 담긴다. 디버그모드에서만 생성)가 생기는데
해더파일과 이 lib파일을 프로젝트 속성의 각 인클루드 링크(추가 종속성까지) 추가 디렉터리에 추가하고 사용하면된다.

6. 주의할점
라이브러리를 만들때 사용할 프로젝트와 개발환경을 맞춰서 빌드한다.
속성->C/C++->코드생성->런다임라이브러리 버전을 맞춰준다. 정적 or 동적 / 디버그 or 릴리즈
